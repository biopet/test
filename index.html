Installation

This tool requires Java 8 to be installed on your device. Download Java 8 here or install via your distribution's package manager.

Download the latest version of ToolTemplate here. To generate the usage run:

java -jar ToolTemplate --version.jar --help

About

This tool is part of BIOPET tool suite that is developed at LUMC by the SASC team. Each tool in the BIOPET tool suite is meant to offer a standalone function that can be used to perform a dedicate data analysis task or added as part of BIOPET pipelines.
